# timer
A simple countdown timer written in Go


## Install

```go get github.com/zealotree/timer/```

## Usage

```$ timer 2s ```

```
 00:00:01
 00:00:00
 Timer completed! 

```

## Play a beep after your timer

``` $timer 2s ; mpv --really-quit beep.ogg ```
